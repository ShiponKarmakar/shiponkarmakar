### 👨‍💻 About Me

I'm a full stack developer and Linux server administrator with a strong focus on WordPress, PHP, and cloud infrastructure. I run **[ZendForce LLC](https://zendforce.com)** — a web development and IT solutions company — and **[UnivaHost](https://univahost.com)**, a web hosting and domain services brand.

I'm an approved **WordPress Core Contributor** with themes and plugins published on WordPress.org. My day-to-day work spans everything from building custom WordPress solutions to provisioning and hardening dedicated Linux servers.

- 🔭 Currently focused on **UnivaHost** and server infrastructure
- 🛠️ Building with **WordPress, Laravel, Vue.js, and LiteSpeed**
- 🌐 Running cPanel/WHM, CloudLinux, CyberPanel, Plesk, and DirectAdmin environments
- 💡 Organizer of the first **IT Professionals Meetup** in Brahmanbaria
- 📫 Reach me at **hello@shiponkarmakar.com**
